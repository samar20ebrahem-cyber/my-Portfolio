# Portfolio Website

A responsive personal portfolio website built from scratch using clean, semantic HTML5, modern CSS3 (Flexbox/Grid), and Vanilla JavaScript. 

This project showcases my frontend development skills, personal projects, and provides a way to get in touch with me.

## Features

- **Responsive Design**: Fully optimized for mobile, tablet, and desktop screens.
- **Dynamic Scroll to Top Button**: Automatically appears after scrolling past the first section to provide a seamless navigation experience.
- **Custom Scrollbar**: Tailored scrollbar design matching the overall color scheme of the website.
- **Interactive Animations**: Hover effects on project cards, smooth scrolling, and dynamic elements.

## Technologies Used

- HTML5
- CSS3 (Custom Variables, Flexbox, Grid)
- Vanilla JavaScript
- Remix Icon (For social and navigational typography iconography)
